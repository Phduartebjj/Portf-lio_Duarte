# Gastar Bilhão / Billion Spender

## BR Sobre o projeto
É um simulador de gastos onde o usuário escolhe um valor inicial e adiciona produtos para visualizar quanto dinheiro ainda resta.


## Objetivo
O projeto foi desenvolvido para praticar JavaScript, Integração com APIs, e lógica de CRUD.

## Funcionalidades
- Definir valor inicial do orçamento(incluindo valores personalizados além de R$ 1 bilhão).
- Adicionar produtos com nome e preço que você queira gastar seu dinheiro.
- Remover produtos da lista.
- Visualizar saldo restante em tempo real

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript 

## Como usar
1. Abra o Projeto.
2. Defina um valor inicial.
3. Adicione produtos existentes ou crie novos produtos.
4. Acompanhe o saldo sendo atualizado automaticamente.

## Melhorias Futuras
- Integração com API de preços.
- Salvamento de dados no LocalStorage.
- Exibir total de gastos e quantidade de produtos comprados.

## US About the project
