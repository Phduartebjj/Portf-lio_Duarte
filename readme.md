### Sobre o projeto
É um simulador de gastos onde o usuário escolhe um valor inicial e adiciona produtos para visualizar quanto dinheiro ainda resta.

### Objetivo
O projeto foi desenvolvido para praticar JavaScript, Integração com APIs, e lógica de CRUD.

### Funcionalidades
- Definir valor inicial do orçamento (incluindo valores personalizados além de R$ 1 bilhão).
- Adicionar produtos com nome e preço que você queira gastar seu dinheiro.
- Remover produtos da lista.
- Visualizar saldo restante em tempo real.

### Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript 

### Como usar
1. Abra o Projeto.
2. Defina um valor inicial.
3. Adicione produtos existentes ou crie novos produtos.
4. Acompanhe o saldo sendo atualizado automaticamente.

### Melhorias Futuras
- Integração com API de preços.
- Salvamento de dados no LocalStorage.
- Exibir total de gastos e quantidade de produtos comprados.

### About the project 
It's a spending simulator, where the user selects an initial amount and adds products to see how much money left.

### Objective 
The project was developed to practice JavaScript, API integration, and CRUD logic.

### Features
- Set an initial balance (Including custom amounts beyond one billion dollars).
- Adds products with custom names and prices what you want to spend your money on.
- Remove products from the list.
- View your remaining balance in real time.

### Tech Stack
- HTML5
- CSS3 
- JavaScript

### How to use
1. Open the Website.
2. Set an initial amount.
3. Add existing products or create new ones.
4. Monitor the balance, which updates automatically.

### Future Enhancements
- Integration with a price API.
- Saving data in localStorage.
- Show total spending and number of products purchased.
